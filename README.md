# Verilog HDL 🚀

A collection of Verilog HDL implementations of fundamental digital logic circuits, written using multiple modeling styles.

## ✨ Features

- Gate Level Modeling
- Dataflow Modeling
- Behavioral Modeling
- Testbenches
- GTKWave Simulation

## 🛠 Tools

- VS Code
- Icarus Verilog
- GTKWave
- Quartus II (FPGA Synthesis)
