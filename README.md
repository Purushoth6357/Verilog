# Verilog HDL 🚀

A collection of **Verilog HDL** implementations of digital logic circuits using **Gate-Level**, **Dataflow**, and **Behavioral** modeling styles.

Each project includes:
- RTL Design
- Testbench
- GTKWave Simulation
- FPGA-ready Verilog Code

---

# 🛠 Tools

- Visual Studio Code
- Icarus Verilog
- GTKWave
- Quartus II Web Edition
- Git & GitHub
